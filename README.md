# y-typescript
基于gulp构建typescript环境

<h3>1、clone本项目</h3>
<pre class="prettyprint linenums" >
 git clone https://github.com/kinghulu/y-typescript.git
</pre>

<h3>2、安装node-sass</h3>
如何<mark>node-sass</mark>安装这里就不介绍了。。

<h3>3、安装依赖包</h3>
执行：
<pre class="prettyprint linenums" >
npm install
</pre>

<h3>4、运行</h3>
<pre class="prettyprint linenums" >
gulp serve
</pre>

如果想只想编译TypeScript，运行
<pre class="prettyprint linenums" >
gulp tsc
</pre>
